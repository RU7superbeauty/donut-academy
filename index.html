01 WHAT IS D0 D0 是什么

D0 是为 AI Agent 原生设计的加密交易 CLI — 不是给人用的 GUI 加了 API，而是从第一行代码开始就为 Agent 而生。

HUMAN INTENT -> AI PARSE -> D0 EXECUTE -> ON-CHAIN SETTLE
CORE CAPABILITIES
  • Zero-Config — 读取操作零配置即用
  • Agent-First — CLI 即 API
  • Non-Custodial — 私钥永远在本地
  • Multi-Exchange — Hyperliquid / Polymarket
WHY IT MATTERS
  • LLM 能力突破 — Agent 理解复杂交易意图
  • 链上基础设施成熟 — 流动性+速度可用
  • CEX 信任危机 — 非托管是确定性趋势
  • 市场空白 — 没有 Agent-native 交易层

DESIGN PRINCIPLES
01

CLI-as-API / 命令行即接口

CLI 本身就是标准接口。AI Agent 直接调用命令行，无需额外 SDK。

02

Zero-Config Progressive Disclosure

80% 功能零 setup。读取操作开箱即用，只有交易才需要私钥。

03

Non-Custodial by Design

架构上不可能接触私钥。EIP-712 签名本地完成。信任来自密码学。

04

Exchange Abstraction

多协议统一命令语义。Agent 不需要知道底层是哪条链、哪个 DEX。

05

Composable Skill System

D0 作为 Clawhub AI Skill 发布，可与其他 Skill 组合成完整交易 pipeline。
