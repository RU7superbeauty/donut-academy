## 交易的第一定律

> **先想清楚最多亏多少，再想可能赚多少。**

95% 的交易者亏钱的原因不是判断错误，而是**单笔亏损过大、无法承受正常的波动**。风险管理是让你在判断失误时还能继续参与游戏的能力。

---

## 核心原则：1-2% 规则

**每笔交易亏损不超过账户总值的 1-2%**。

```
账户 $1,000 × 1% = $10 最大亏损/笔
账户 $1,000 × 2% = $20 最大亏损/笔

即使连续亏损10笔（极端情况）：
账户剩余 = $1,000 × (1-0.01)^10 ≈ $904
→ 只亏了 9.6%，完全可以恢复
```

违反这个规则（每笔亏 10%）：

```
连续亏损10笔：$1,000 × (1-0.10)^10 ≈ $349
→ 亏了 65%，几乎不可能翻回来
```

## 止损：必须设置，没有例外

止损是你在建仓时就确定的价格，**一旦触发就无条件平仓**。

```bash
# 建仓后立即设置止损
d0 hl:limit buy ETH 0.1 3200     # 建仓
d0 hl:stop-loss ETH 0.1 3040     # 止损在 -5%
```

**止损放哪里**：

```
方法 1：技术支撑位下方
  最近支撑位 $3,100 → 止损设在 $3,060（支撑下 1.3%）
  
方法 2：固定百分比
  入场价 $3,200 × (1 - 5%) = $3,040
  适合初学者，简单清晰

方法 3：ATR（平均真实范围）
  止损 = 入场价 - 2×ATR
  更科学，但 d0 ta 目前直接给出建议止损位
```

## 风险回报比（R:R Ratio）

每笔交易前必须确认风险回报比：

```
R:R = 预期收益 / 最大亏损

入场：$3,200
止损：$3,040（-5%，亏 $160）
目标：$3,680（+15%，赚 $480）

R:R = $480 / $160 = 3:1
```

**最低标准：R:R ≥ 2:1**

```
如果 R:R = 1:1，胜率需要 > 50% 才能盈利
如果 R:R = 2:1，胜率只需 > 33% 就能盈利
如果 R:R = 3:1，胜率只需 > 25% 就能盈利

→ 高 R:R 是业余者向专业者进化的核心区别
```

## 仓位大小计算

已知最大亏损金额和止损距离，反推仓位大小：

```
仓位大小 = 最大亏损金额 ÷ 止损距离

账户 $1,000，每笔最多亏 $20（2%）
入场价 $3,200，止损 $3,040（距离 $160）

仓位大小 = $20 / $160 = 0.125 ETH

验证：0.125 ETH × $160 止损距离 = $20 ✓
```

```bash
# 实际执行
d0 hl:limit buy ETH 0.125 3200
d0 hl:stop-loss ETH 0.125 3040
```

## 杠杆的正确使用方式

杠杆不是"让我买更多"，而是**资本效率工具**：

```
不用杠杆：买 0.1 ETH 需要 $320 保证金
用 2x 杠杆：买 0.1 ETH 只需要 $160 保证金
→ 省出的 $160 可以用于其他机会

错误用法：同样的保证金，买 0.2 ETH（加倍风险）
正确用法：同样的保证金，买 0.1 ETH，省出资金做其他配置
```

```bash
# 设置杠杆（资金管理类交易建议 1-3x）
d0 hl:leverage ETH 2

# 高杠杆（5x+）只适合以下情况：
# - 止损非常近（< 2%）
# - 仓位极小
# - 有明确催化剂
```

## Kelly 公式：科学确定仓位

凯利公式（Kelly Criterion）是数学上最优的仓位计算方法：

```
Kelly% = W - (1-W)/R

W = 胜率（过去交易的赢的比例）
R = 平均盈亏比（平均赢 / 平均亏）

例：胜率 55%，盈亏比 2:1
Kelly% = 0.55 - (0.45/2) = 0.55 - 0.225 = 32.5%

实际使用 Half-Kelly（÷2）= 16.25%
→ 每笔交易用账户 16% 建仓
```

> **建议**：初学者不要使用 Full Kelly，Half Kelly 或 Quarter Kelly 更安全。没有足够历史数据前，保守使用 1-5% 仓位。

## 常见致命错误

| 错误 | 后果 | 正确做法 |
|---|---|---|
| 不设止损 | 单笔亏光账户 | 建仓同时设止损，无例外 |
| 移动止损（往不利方向）| 越亏越深 | 止损只能向有利方向移动 |
| 加仓摊低成本 | 扛着亏损仓位越来越重 | 有盈利才加仓，亏损减仓 |
| 仓位过重 | 心态崩溃，无法理性决策 | 单笔不超过账户 5% |
| 频繁交易 | 手续费蚕食利润 | 有明确信号才入场 |

## 动手练习

```bash
# 计算你的第一笔交易参数
# 假设账户 $200，每笔最多亏 $4（2%）

# 查看 ETH 价格
d0 price ETH

# 查看技术面
d0 ta ETH

# 根据分析确定：
# 1. 入场价
# 2. 止损价（技术支撑下方或 -5%）
# 3. 目标价（R:R 至少 2:1）
# 4. 仓位大小 = $4 / 止损距离

# 执行（先用最小仓位练习）
d0 hl:leverage ETH 2
d0 hl:limit buy ETH <计算出的仓位> <入场价>
d0 hl:stop-loss ETH <仓位> <止损价>
d0 hl:positions
```

## 核心概念回顾

- **1-2% 规则**：单笔最大亏损不超过账户 1-2%
- **止损**：建仓同时设置，无例外，不向不利方向移动
- **R:R**：风险回报比至少 2:1，3:1 更好
- **仓位大小** = 最大亏损 ÷ 止损距离
- **Kelly 公式**：科学仓位计算，实际用 Half-Kelly
