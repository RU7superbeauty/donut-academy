## 为什么技术分析仍然有效？

技术分析不是玄学，它是**自我实现的预言**。当足够多的交易者相信"价格在 $3,200 支撑位会反弹"，他们就会在这里买入，从而真正造成反弹。

D0 的 `d0 ta` 命令整合了多个技术指标，让你在1秒内获得人类分析师需要数分钟的信息。

```bash
d0 ta ETH
```

```
ETH/USDC  $3,247.50
────────────────────────────────
RSI(14):     34.2  ← oversold
MACD:        bullish crossover forming
MA20:        $3,180  (price above)
MA50:        $3,050  (price above)
Support:     $3,100 / $2,950
Resistance:  $3,400 / $3,600
Signal:      LONG  (confidence: 62%)
```

## K 线基础

每根 K 线代表一段时间内的价格运动：

```
        ┃  ← 上影线（最高价）
   ┌────┸────┐
   │  实体   │  ← 开盘价到收盘价
   │（绿色=涨）│
   └────┰────┘
        ┃  ← 下影线（最低价）

绿色 K 线：收盘价 > 开盘价（上涨）
红色 K 线：收盘价 < 开盘价（下跌）
```

**关键形态识别**：

| 形态 | 含义 | 可信度 |
|---|---|---|
| 长下影线（锤子线）| 买方力量强，潜在底部 | ⭐⭐⭐ |
| 长上影线（射击之星）| 卖方压力大，潜在顶部 | ⭐⭐⭐ |
| 吞没形态（两根K线）| 趋势反转信号 | ⭐⭐⭐⭐ |
| 十字星 | 多空平衡，方向待定 | ⭐⭐ |

## RSI：超买超卖的量化

RSI（相对强弱指数）衡量近期涨跌幅度，范围 0-100：

```
RSI > 70：超买区间 → 价格可能回落
RSI < 30：超卖区间 → 价格可能反弹
RSI 40-60：中性区间 → 无明显信号
```

**实际应用**：

```bash
d0 ta BTC
# RSI(14): 28.5 → 超卖，关注反弹机会

d0 ta ETH  
# RSI(14): 72.1 → 超买，注意回调风险
```

> **重要**：RSI 在强趋势中可以长期保持极值。超卖不等于"一定反弹"，需要配合其他信号。

## MACD：趋势动量指标

MACD 由两条均线的差值组成，用于判断趋势方向和动量：

```
MACD线 = 12日EMA - 26日EMA
信号线 = MACD线的9日EMA
柱状图 = MACD线 - 信号线

金叉（MACD上穿信号线）→ 买入信号
死叉（MACD下穿信号线）→ 卖出信号
```

**D0 ta 的 MACD 输出解读**：

```
MACD: bullish crossover forming  → 金叉即将形成，看多
MACD: bearish crossover          → 死叉已形成，看空
MACD: divergence detected        → 价格创新高但MACD未跟随，警惕顶背离
```

## 均线：趋势的骨架

均线（MA）平滑价格，过滤短期噪音：

```
MA20 (20日均线)：短期趋势
MA50 (50日均线)：中期趋势
MA200 (200日均线)：长期趋势

黄金交叉：MA20 上穿 MA50 → 看多
死亡交叉：MA20 下穿 MA50 → 看空
价格在 MA200 上方：长期多头趋势
```

**判断框架**：

```bash
d0 ta ETH

# 检查：
# 价格 vs MA20/MA50 的位置
# 均线排列（MA20 > MA50 > MA200 = 多头排列）
# 均线斜率（向上 = 多头，向下 = 空头）
```

## 支撑与阻力

支撑和阻力是价格反复测试的关键价位：

```
阻力位：价格多次触及但未能突破的上方价位
支撑位：价格多次触及但未能跌破的下方价位

突破阻力 → 原阻力变支撑（角色互换）
跌破支撑 → 原支撑变阻力（角色互换）
```

**D0 直接给出关键价位**：

```bash
d0 ta ETH
# Support:    $3,100 / $2,950
# Resistance: $3,400 / $3,600
```

## 综合使用：5步分析框架

```bash
d0 ta ETH
```

拿到结果后，按这个顺序分析：

```
1. 价格位置：在均线上方还是下方？
   → 上方 = 多头偏向；下方 = 空头偏向

2. RSI 值：超买/超卖/中性？
   → 极值区间 + 反转K线 = 高置信信号

3. MACD：金叉/死叉/背离？
   → 叉的方向 = 短期动量方向

4. 关键价位：距离支撑/阻力多远？
   → 接近支撑做多胜率更高

5. Signal 综合评分：LONG/SHORT/NEUTRAL
   → 置信度 > 60% 时值得关注
```

## 核心概念回顾

- **RSI**：超卖 < 30，超买 > 70，配合其他信号使用
- **MACD**：金叉看多，死叉看空，背离是警告
- **均线**：趋势方向指示，多头排列更安全做多
- **支撑/阻力**：关键价位，突破确认后角色互换
- **d0 ta**：一条命令获取完整技术分析
